Zero-server by design.

Content scripts normalize chats, the background worker writes IndexedDB rows, and the options UI handles vault browsing, local extraction, export, sync, and backup.

15 chat platforms

ChatGPT, Claude, Gemini, Perplexity, DeepSeek, Grok, Copilot, Mistral, Poe, Kimi, Qwen, ChatGLM, Doubao, NotebookLM.

Content scripts

Adapters normalize chat ID, title, URL, model, and messages.

Background worker

Capture trigger, throttle, dedupe, diagnostics, and export downloads.

IndexedDB vault

Chats, messages, folders, open threads, extraction runs, and search index.

Vault UI

Search, filter, pin, folders, tags, chat detail, source links, settings, diagnostics, and export history.

Local extraction

Candidate filtering, chunking, Transformers.js WebGPU/WASM, and optional Chrome Prompt API.

Export and sync

JSON, CSV, TSV, Markdown, offline HTML, print PDF, Obsidian sync, ZIP fallback, and encrypted backups.

Privacy boundary

No Rakuzaichi server. Captured chats, extracted threads, search indexes, settings, backups, and sync handles stay on-device.

Rakuzaichi runs locally. No telemetry, no analytics, no Rakuzaichi server.
