# @dorso/cli

CLI companion for Dorso's local status export.

## Install

```sh
npm install -g @dorso/cli
```

## Usage

```sh
dorso status
dorso status --json
dorso status --prompt
dorso status --watch
dorso status --path ~/Downloads/dorso/status.json
```

Set `DORSO_STATUS_PATH` to avoid passing `--path`.
